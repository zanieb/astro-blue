Astro Blue includes everything from Starlight plus custom enhancements.

## Starlight Features

All standard Starlight features are available:

- **Full-text search** — Built-in Pagefind search
- **Internationalization** — Multi-language support
- **SEO optimized** — Automatic meta tags and sitemap
- **Responsive design** — Works on all screen sizes
- **Accessibility** — WCAG compliant

## Astro Blue Enhancements

### Astral Color Palette

Custom CSS variables with the official Astral brand colors:

- Deep purple primary (`rgb(38 18 48)`)
- Lime accent (`rgb(215 255 100)`)
- Refined grayscale from the Astral design system

### Mintlify-Style Sidebar

- Clean, minimal navigation
- Uppercase section headers
- No visual clutter
- Proper nested hierarchy

### Dark Mode First

Optimized for dark mode viewing with carefully tuned contrast ratios.

See [Components](/concepts/components/) for all available components including:

- Admonitions (Note, Tip, Caution, Danger)
- Code blocks with syntax highlighting
- Cards and card grids

- Steps
