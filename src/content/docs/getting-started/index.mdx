Welcome to Astro Blue! This section will help you get up and running quickly.

## What is Astro Blue?

Astro Blue is a documentation theme built on [Starlight](https://starlight.astro.build/) with custom styling inspired by the Astral design system. It features:

- **Astral color palette** with deep purple and lime accent colors
- **Clean sidebar navigation** with proper hierarchy
- **Dark mode first** design with light mode support
- **All Starlight components** including Cards, Tabs, Asides, and more
