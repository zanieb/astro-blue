Welcome to **Astro Blue** — a customized documentation theme built on Starlight.

## Features

1. **Dark mode first** — A refined dark theme with custom accent colors
2. **Clean navigation** — Organized sidebar with semantic grouping
3. **Modern typography** — Optimized for readability

## Getting Started

```bash
bun install
bun run dev
```

Visit `http://localhost:4321` to see your docs.
