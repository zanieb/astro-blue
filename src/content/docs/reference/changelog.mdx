## v1.0.0

Initial release of Astro Blue.

### Features

- Astral color palette integration
- Mintlify-style sidebar
- Dark mode first design
- All Starlight components supported

### Components

- Custom asides styling
- Enhanced code blocks
- Styled cards and card grids
- Synced tabs support
- Steps component

### Configuration

- Simple `astro.config.mjs` setup
- CSS variable customization
- Sidebar configuration options
