Deploy your documentation site to Vercel with zero configuration.

## Quick Deploy

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel auto-detects Astro and configures the build

## Manual Configuration

Create `vercel.json` in your project root:

```json
{
  "buildCommand": "bun run build",
  "outputDirectory": "dist",
  "framework": "astro"
}
```

## Environment Variables

Set environment variables in the Vercel dashboard or via CLI:

```bash
vercel env add MY_VARIABLE
```

## Preview Deployments

Every pull request gets a preview deployment automatically.
