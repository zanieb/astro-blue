Astro Blue uses the official Astral color palette from pyx.dev.

## Primary Colors

| Variable | Value | Usage |
|----------|-------|-------|
| `--astral-primary` | `rgb(38 18 48)` | Deep purple, selected states |
| `--astral-primary-light` | `rgb(215 255 100)` | Lime accent, hover states |
| `--astral-primary-dark` | `rgb(99 64 172)` | Purple accent |

## Background Colors

| Variable | Value | Usage |
|----------|-------|-------|
| `--astral-bg-light` | `rgb(255 255 255)` | Light mode background |
| `--astral-bg-dark` | `rgb(13 14 13)` | Dark mode background |

## Using Colors

Reference colors in your custom CSS:

```css
.my-element {
  background: var(--astral-primary);
  color: var(--astral-primary-light);
}
```

## Starlight Mappings

The Astral palette maps to Starlight variables:

```css
:root {
  --sl-color-accent-low: var(--astral-primary);
  --sl-color-accent: var(--astral-primary-dark);
  --sl-color-accent-high: var(--astral-primary-light);
  --sl-color-black: var(--astral-bg-dark);
}
```
