**Astro Blue** is a documentation theme built on [Starlight](https://starlight.astro.build/) with custom styling inspired by the Astral design system.

## Features

- **Astral color palette** — Deep purple and lime accent colors from the official Astral brand
- **Mintlify-style sidebar** — Clean, minimal navigation with proper hierarchy
- **Dark mode first** — Optimized for dark mode with light mode support
- **All Starlight components** — Full access to Cards, Tabs, Asides, and more

## Quick Start

```bash
# Clone the template
git clone https://github.com/example/astro-blue
cd astro-blue

# Install dependencies
bun install

# Start development server
bun run dev
```
