title: Tabs

import { Tabs, TabItem } from '@astrojs/starlight/components';

Use tabs to organize content into multiple panels that users can switch between. Tabs are ideal for showing different options or versions, such as package managers or operating systems.

## Basic Tabs

<Tabs>
  <TabItem label="npm">```bash npm install @astrojs/starlight ```</TabItem>
  <TabItem label="pnpm">```bash pnpm add @astrojs/starlight ```</TabItem>
  <TabItem label="bun">```bash bun add @astrojs/starlight ```</TabItem>
</Tabs>

```mdx
<Tabs>
  <TabItem label="npm">Content for npm tab</TabItem>
  <TabItem label="pnpm">Content for pnpm tab</TabItem>
</Tabs>
```

## Synced Tabs

Tabs with the same `syncKey` stay synchronized:

<Tabs syncKey="package-manager">
  <TabItem label="npm">npm install</TabItem>
  <TabItem label="bun">bun add</TabItem>
</Tabs>

Selecting a tab here will also switch the tab below:

<Tabs syncKey="package-manager">
  <TabItem label="npm">npm run dev</TabItem>
  <TabItem label="bun">bun run dev</TabItem>
</Tabs>

```mdx
<Tabs syncKey="package-manager">
  <TabItem label="npm">...</TabItem>
  <TabItem label="bun">...</TabItem>
</Tabs>
```

## Properties

### Tabs

| Property  | Type     | Description                                                                                         |
| --------- | -------- | --------------------------------------------------------------------------------------------------- |
| `syncKey` | `string` | Optional key to synchronize multiple tab groups. Tabs with the same `syncKey` will switch together. |

### TabItem

| Property | Type     | Description                                                                                |
| -------- | -------- | ------------------------------------------------------------------------------------------ |
| `label`  | `string` | **Required.** The text displayed on the tab.                                               |
| `icon`   | `string` | Optional icon name from [Starlight icons](https://starlight.astro.build/reference/icons/). |
