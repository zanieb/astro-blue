## Installation

<Steps>

1. **Install the package**

   ```bash
   npm install @acme/sdk
   ```

2. **Set up your API key**

   Get your API key from the [dashboard](https://dashboard.example.com) and add it to your environment:

   ```bash
   export ACME_API_KEY="your_api_key_here"
   ```

3. **Initialize the client**

   ```typescript
   import { createClient } from '@acme/sdk';

   const client = createClient({
     apiKey: process.env.ACME_API_KEY,
   });
   ```

4. **Make your first request**

   ```typescript
   const users = await client.users.list();
   console.log(users);
   ```

</Steps>

## Full Example

Here's a complete example that demonstrates the main features:

```typescript
import { createClient } from '@acme/sdk';

async function main() {
  // Initialize client
  const client = createClient({
    apiKey: process.env.ACME_API_KEY,
  });

  // List users
  const users = await client.users.list({ limit: 10 });
  console.log(`Found ${users.length} users`);

  // Create a user
  const newUser = await client.users.create({
    name: 'Alice',
    email: 'alice@example.com',
  });
  console.log('Created user:', newUser.id);

  // Update a user
  await client.users.update(newUser.id, {
    name: 'Alice Smith',
  });

  // Delete a user
  await client.users.delete(newUser.id);
}

main().catch(console.error);
```
