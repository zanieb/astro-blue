# Header Differences: PYX (Mintlify) vs Astro Blue

Comparison of header/navigation between PYX Documentation (https://docs.pyx.dev) and our Astro Blue theme.

---

## Site Title / Logo

| Feature       | PYX (Mintlify)                            | Astro Blue                 | Priority |
| ------------- | ----------------------------------------- | -------------------------- | -------- |
| Title format  | "PYX" bold + "Documentation" lighter gray | "Astro Blue" single weight | High     |
| Title spacing | More breathing room around logo           | Tighter                    | Medium   |

---

## Search Bar

| Feature           | PYX (Mintlify)                                        | Astro Blue              | Priority |
| ----------------- | ----------------------------------------------------- | ----------------------- | -------- |
| Width             | Very wide (~350px), takes up significant header space | Narrower (~200px)       | High     |
| Centering         | Truly centered in header                              | Left of center          | High     |
| Placeholder       | "Search..." with ellipsis                             | "Search"                | Low      |
| Keyboard shortcut | ⌘K badge inside search bar, right-aligned             | ⌘K badge                | Low      |
| Background        | Light gray fill                                       | Transparent with border | Medium   |

---

## Navigation Links

| Feature      | PYX (Mintlify)                         | Astro Blue        | Priority |
| ------------ | -------------------------------------- | ----------------- | -------- |
| Text links   | "Contact", "Service Status"            | None (only icons) | Medium   |
| CTA button   | Purple "Dashboard >" button with arrow | None              | Low      |
| Social icons | None visible in header                 | GitHub icon       | Low      |

---

## Header Layout

| Feature          | PYX (Mintlify)              | Astro Blue   | Priority |
| ---------------- | --------------------------- | ------------ | -------- |
| Bottom border    | Very subtle horizontal rule | None         | High     |
| Vertical padding | More generous (~16px)       | Tighter      | Medium   |
| Overall height   | Taller, more spacious       | More compact | Medium   |

---

## Theme Toggle

| Feature    | PYX (Mintlify) | Astro Blue         | Priority |
| ---------- | -------------- | ------------------ | -------- |
| Icon style | Sun/moon icon  | Gear/settings icon | Medium   |
| Position   | Far right      | Far right          | -        |

---

## Summary of High Priority Items

1. **Site title format** - Add "Documentation" in lighter weight after site name
2. **Search bar width** - Make search bar significantly wider
3. **Search bar centering** - Center the search bar in the header
4. **Header bottom border** - Add subtle horizontal rule under header
